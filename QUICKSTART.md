# 🚀 Quick Start Guide

## Three Commands to Get Running

### 1. Get Dependencies
```bash
flutter pub get
```

### 2. Generate Hive Adapters (REQUIRED!)
```bash
flutter pub run build_runner build --delete-conflicting-outputs
```

### 3. Run the App
```bash
flutter run
```

---

## That's it! 🎉

The app will launch and you can start:
1. Creating domains
2. Adding tasks
3. Tracking your progress
